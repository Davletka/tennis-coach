# Screenshots

Auto-generated screenshots of the CourtCoach frontend.

## Captured scenarios

| Scenario | Viewports |
|----------|-----------|
| `01-analyze-idle` | desktop (1440×900), mobile (390×844) |
| `02-history-signed-out` | desktop, mobile |
| `03-compare-signed-out` | desktop, mobile |
| `04-progress-signed-out` | desktop, mobile |

## Regenerate

```bash
npm run screenshots
```

The script auto-starts the Next.js dev server if none is running on port 3000, captures all scenarios, then shuts it down. PNG files are git-ignored — only this README is committed.
